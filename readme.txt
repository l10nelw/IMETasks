IMETasks.bat puts together various day-to-day Internet Monitoring Engineer tasks together in one interface, including:
- run HTTPConnector service
- run URLFileFilter application
- access various network config files
- search through local config files
- back up Harvest Sheet and open last backed up Harvest Sheet
- update IMETasks when new version available

IMETasks.bat requires certain settings be set up in IMETasks.ini.

Support script listjob.py included; requires Python 3.
